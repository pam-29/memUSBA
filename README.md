Connaissez-vous les memes ?
Ces images légendées d’une blague sont devenues très populaires auprès de la nouvelle génération. Nous avons donc décidé de se servir de ce format pour faire découvrir des œuvres de l’aile Bonheur du Musée des Beaux Arts de Bordeaux.
Ainsi, l’utilisateur aura la possibilité de choisir un tableau parmi 5 - proposées aléatoirement parmi un ensemble de 23 tableaux, de créer son propre meme puis de voter pour son préféré parmi ceux créés par les visiteurs.



Stack utilisé
-Laravel (php)
-Blade
-JavaScript
